## About

ChilliTech is a multi-tenant SaaS platform that digitizes operations of agricultural cold storage businesses. Farmers interact via WhatsApp; operators use a dashboard. This repo holds design-stage documentation for stakeholder review.

> This is a design/documentation repo. Application source code lives in a separate private repository.
